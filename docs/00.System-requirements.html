1  System requirements

1.1 Main system requirements

  • Ubuntu 18.04
  • R 4.3.0, Bioconducctor 3.17
  • Python 3.7

1.2 Non-standard hardware requirements

A RTX 2080 Ti GPU was used to improve the training speed of convolutional neural neworks.

1.3 Data processing softwares

1.3.1 Processing tools

  • Bowtie2 2.4.5
  • samtools 1.9
  • deeptools 2.26.0
  • bedtools 2.29.1
  • MACS2 2.2.7.1
  • seqkit 2.2.0
  • Keras
  • Tensorflow

1.3.2 Processing workflow managers

Most sequencing data was processed with tinyMapper 0.11.3-0.12.14.
Hi-C data was processed with hicstuff 3.1.2 and Chromosight 1.4.1.
